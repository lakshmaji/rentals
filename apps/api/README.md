# Rentals API

## Description
 
 The API are built using [Nest](https://github.com/nestjs/nest) framework.

## Installation

```bash
$ yarn install
```

## Environment variables

```bash
$ cp .env.example .env
```

## Documentation 

```bash
$ yarn docs
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
